We use the **3–5 Rule** to decide the session start time based on how many players are present.
- **3 players** → Session runs from **3:00 to 3:30**
- **4–5 players** → Session runs from **3:30 to 4:00**

This rule helps keep sessions consistent and prevents the game from feeling rushed. It also helps the [[Dungeon Master]] prepare properly, since not every session can be planned far in advance.

The rule is used when the group normally has **5 players total** (Or any campaign that have mostly 5 constant players), so no one misses too much of the story if someone is absent.
