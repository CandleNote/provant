Whenever you flank a creature which must be a straight opposite line where for example it look like this :

|     | **Ally** |     | **Ally** |       |          |
| --- | -------- | --- | -------- | ----- | -------- |
|     | Enemy    |     |          | Enemy |          |
|     | **Ally** |     |          |       | **Ally** |
|     |          |     |          |       |          |
You will gain the flank condition where you gain a +3 towards your attack roll, and +1 towards your damage roll. This effect everything that has an attack roll in it.
